INSTALLATION--------------------------------------------------
1) Please Un-zip the file into your local drive.
2) In your favourite terminal such as cmd, Powershell, or the linux command line, navigate to the file and open the CarParkSystem folder.
3) Within this folder, type the command 'npm install'. This will install all of the dependencies necesary for the application to run.
4) Once installed, type 'node app.js'. This will open up the server.
5) Please go to "localhost:3000" in your favourite browser where you will be redirected to the homepage.
